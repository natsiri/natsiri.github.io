Parentheses ( )

Parentheses are used commonly in coding and they are used to contain a list of parameters passed to functions and control strutures. In some cases there may be no parameters and the Parentheses are left blank like so ()

Brackets [ ]

Brackets are used in what we call Arrays which is used to store a large amount of data such as exampleArray[1,2,3,4,5] this is not limited to numbers but can have many other types such as Strings.

Braces { }

Braces are used in JavaScript also very commonly as it is used to create objects.

Single quotes ' ' and Double quotes " "

Both single and double quotes are used in JavaScript to write a String such as so "String test" or 'String test2' both can be used as it results are the same but is best to keep the code clean by using 1 type consisitly
