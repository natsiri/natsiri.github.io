What a responsive site is, and why responsiveness is important

A responsive site is when a page can be opened on multiple devices and be automaticly resized to the screen size. It's important as now adays many people will have different types of devices where they would access sites such as iphones,tablets, and computers.

What frameworks are, and their pros and cons

Framework is where a structure is given to you where you can code of from. Such as basic code within a file where you would add data into where it would simpfy the process. Therefore you can focus on the bigger parts of the project. Pros - Don't have to start from scratch, reuseable Cons - Some frameworks can't be modified.

What a wireframe is and why we use it

Wireframe is a sketch of your page layout such as a webpage but by only using simple lines (wires). We use it so that we can get a basic draft idea down of our plan.
